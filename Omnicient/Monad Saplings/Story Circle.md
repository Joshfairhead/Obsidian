# Story Circle

- Story circle 
	- You (comfort)
	- Need (they want)
	- Go (unfamiliar situation)
	- Search (adapts to situation)
	- Find (gets whats wanted)
	- Take (pay a price)
	- Return (to familiar situation)
	- Change (having changed)
